## Clustering Algorithm

每个聚类算法都有两个元素：一个类，实现训练方法；一个函数，输入训练数据，返回整型标签数组，标签表示不同的聚簇。

1. KMeans
2. MeanShift
3. Agglomerative
4. DBSCAN
5. SpectralClustering
6. Birch
7. AffinityPropagation


## Input Format
- one vector for one line

- number of vectors

- number of cluster's 

## Output Format
%d(%d) - 
[Num. of vector], [Num. of cluster]
